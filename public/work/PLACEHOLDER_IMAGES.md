## Image Guidelines

1. **Format**: Use JPG for photos, optimize for web
2. **Quality**: Export at 80-85% quality for balance of size/quality
3. **Resolution**: 2x the display size for retina displays
4. **Orientation**: Match the aspect ratios listed above for best results
5. **Style**: Consistent color grading across all images looks most professional
